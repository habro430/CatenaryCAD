Синтаксис
[Serializable]
public class ConcurrentHashSet<T> : IReadOnlyCollection<T>, ICollection<T>, IEnumerable<T>, IEnumerable
Параметры Типа
Имя Описание
T

Тип объектов в коллекции.
Примечание

Все открытые члены ConcurrentHashSet<T> являются потокобезопасными и могут использоваться одновременно из нескольких потоков.
