MemoryCache(Double, Double)

Инициализирует новый экземпляр класса MemoryCache<T>.

Объявление
public MemoryCache(double sliding, double absolute)
Параметры
Тип Имя Описание
System.Double sliding

Значение времени (в минутах), по истечению которого, объект удаляется из кэша, если к нему за это вермя не было обращений.
System.Double absolute

Значение времени (в минутах), по истечению которого, объект удаляется из кэша, независимо от sliding.
Примечание

Если требуеться указать что записи в кэше не должны иметь скользящего и/или абсолютного срока дейсвия, то аргументам sliding и absolute необходимо присвоить 0.

Исключения
Тип Состояние
System.ArgumentOutOfRangeException

sliding меньше чем 0.
System.ArgumentOutOfRangeException

absolute меньше чем 0.
